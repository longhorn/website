Enterprise Edition

Longhorn is a CNCF project — a 100% open source, cloud-native distributed storage system for Kubernetes. It is designed to be vendor-neutral and community-driven, empowering users to run reliable, lightweight, and easy-to-manage persistent storage across any Kubernetes environment.

The listed partners offer enterprise distributions, training, and commercial support for Longhorn:

SUSE Storage – Enterprise Solution for Longhorn

SUSE Storage is an enterprise distribution built on Longhorn — combining the simplicity and agility of the community project with SUSE’s proven enterprise expertise. It provides production-ready support, performance optimizations, and lifecycle management for clusters running in demanding environments. SUSE Storage creates a dedicated storage controller for each block device volume and synchronously replicates the volume across multiple replicas stored on multiple nodes. The storage controller and replicas are themselves orchestrated using Kubernetes, ensuring resilience and scalability.

  • Enterprise Support: Backed by SUSE’s global support organization for mission-critical workloads.
  • Security and Compliance: Regularly tested and updated for security compliance and reliability.
  • Certified Integrations: Fully validated with SUSE’s cloud-native ecosystem components.
  • Lifecycle Management: Simplified installation, upgrade, and monitoring workflows with SUSE tooling.

Vendor Participation

The Longhorn project welcomes participation from any organization providing enterprise services, distributions, or commercial support built around Longhorn. If your company offers an enterprise-grade Longhorn-based solution, you are encouraged to contribute and be listed on this page.

About Longhorn

Longhorn remains a vendor-neutral, open-source project under the Cloud Native Computing Foundation (CNCF).Its success is driven by contributions from individuals, users, and organizations worldwide. Enterprise distributions such as SUSE Storage are built on top of Longhorn, helping expand its adoption and reliability in production environments — while the upstream community continues to innovate openly.
